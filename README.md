# GrowthReady Algebra

Private, local-first teacher dashboard for turning MyOpenMath Algebra 1 diagnostic CSV exports into instructional decisions.

## Version 1 privacy boundaries

- CSV upload only.
- No MyOpenMath scraping or automated login.
- No database, authentication, cloud sync, paid service, or external AI API.
- Student data is parsed, analyzed, and stored locally in the browser.

## Commands

```sh
npm install
npm run dev
npm run build
npm test
```
